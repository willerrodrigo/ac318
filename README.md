# Green Market

Green Market is a market box front application to reduce the consumption of plastic bags.

## Prerequisites

Create an .env file in the root of the backend directory and set the environment variables, as in the .env.example file.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install Green Market dependencies.

```bash
npm install
```

## Usage

```bash
BackEnd:
	npm run dev

FrontEnd:
	npm start
```
